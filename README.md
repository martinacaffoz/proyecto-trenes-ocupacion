# 🚆 Proyecto: Ocupación de trenes en tiempo real

## 📌 Descripción
Este proyecto simula un sistema que permite visualizar el nivel de ocupación de los vagones de un tren.

El objetivo es mejorar la experiencia de los usuarios, permitiéndoles elegir el vagón menos congestionado.

## ⚙️ Funcionalidades
- Simulación de ocupación de vagones
- Visualización de porcentajes
- Recomendación del mejor vagón

## 🧠 Idea
Surge a partir de la experiencia cotidiana en transporte público, donde la distribución de pasajeros no es eficiente.
